HTML Workflow
==================
Download:
- Node.js: https://nodejs.org/en/download/ 
- Git: https://git-scm.com/downloads

Dependencies:
- node / npm
- git
- gulp

Install gulp version 4.0.0:
```sh
$ npm install gulp -g
```

Clone the repository:    
```sh
$ git clone https://github.com/ryahpalma/html-workflow.git
```
Enter in the folder:
```sh
$ cd html-workflow
```
Install dependencies:
```sh
$ npm install
```
Start gulp
```sh
$ gulp
```
