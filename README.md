HTML Workflow
==================
Download:
- Node.js: https://nodejs.org/en/download/ 
- Git: https://git-scm.com/downloads

Dependencies:
- node / npm
- git
- gulp

Install gulp:
```sh
$ npm install gulp -g
```

Clone the repository:    
```sh
$ git clone https://github.com/ryahpalma/htmlworkflow.git
```
Enter in the folder:
```sh
$ cd htmlworkflow
```
Install dependencies:
```sh
$ npm install
```
Start gulp
```sh
$ gulp
```
